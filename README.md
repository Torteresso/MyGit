### MGIT

A partial implementation of the Git version control in Kotlin.

### Install

Go to the repository that you want to version control.
And run the installHere script : 

```bash
cd someRepoToVersionControl
bash path/to/MyGit/installHere.sh
```

This will generate a ./mgit executable in the current repository.

### Usage

Then simply run  :

```bash
# Show available commands
./mgit --help 

# Show specific command usage
./mgit init --help
```
